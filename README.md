## Install project

Requirements

PHP 7.4+
MySQL 5.7+
NodeJS
NPM
Yarn

1- git clone https://github.com/alfjuniorbh/keyassist
2- cd keyassist
3- create database
4- composer install
5- npm install
6- npm run dev
7- copy .env and set database informations
8- php artisan key:generate
9- php artisan migrate
10- php artisan server
